# O-uso-de-intelig-ncia-artificial-para-classifica-o-d-eimagens-de-sat-lite-no-estado-do-Par-.
Este repositório apresenta os códigos usados para obter os resultados apresentados no TCC descrito pelo titúlo.

O primeiro passo é ter a base de dados matricial e esses podem ser obtidos usando o Cod_GEE.
Google Earth Engine (JavaScript) – Processamento e Análise de Dados Geoespaciais
Desenvolvimento de scripts em JavaScript para processamento e análise de imagens de satélite em nuvem.
Manipulação de grandes conjuntos de dados geoespaciais, incluindo imagens de sensores orbitais (ex.: Sentinel e Landsat).
Aplicação de técnicas de sensoriamento remoto, geração de índices espectrais (como NDVI), classificação e análise temporal da cobertura do solo.
Produção de mapas temáticos e informações espaciais para apoio à tomada de decisão.

A partir dele pode-se gerar amostras independentes ou usar o csv anexado.

Para classificar suas imagens, basta utilizar o modelo_treiando.
